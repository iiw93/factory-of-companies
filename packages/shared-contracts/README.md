# Shared Contracts

Пакет содержит машиночитаемые контракты Bridge Layer для команд и ответов.

Документы в `docs/specs` являются смысловым источником. Schema-файлы в этом пакете являются формальной машинной проекцией этих документов.

Изменения schema должны оставаться согласованными с `docs/specs/bridge-command-contract.md`, `docs/specs/bridge-response-contract.md` и acceptance checklist в `tests/acceptance`.
